# 反应表

> 原文:[https://www.tutorialandexample.com/react-table/](https://www.tutorialandexample.com/react-table/)

**反应表**

该表以**行**和**列**的形式组织信息。该表主要用于以结构化格式存储和显示数据。

### 反应表的功能

下面列出了**反应表**的几个特征:

*   它重量很轻，大约为。11 kb(样式只需要 2kb)。
*   它是可定制的。
*   它可以通过使用可选的道具和回调来控制。
*   它包括客户端和服务器端分页。
*   它还包括过滤器。
*   它有一个最小的设计，也很容易主题化。
