**Selenium WebDriver-CSS 选择器定位器**

CSS 选择器用于根据 web 元素的**“# id”**和**来识别 web 元素。类”、Html 标签和属性。**

CSS 代表**级联样式表，**用于显示用 Html 或 xml 语言编写的文档。

CSS 选择器比 XPath 更快。
