### 缓冲阅读器

*   使用这个类，我们可以从输入流中读取数据。
*   该类继承 Reader 类的属性。

 **### read()方法

*   它用于从流中读取单个字符。
*   如果到达输入的末尾，它将返回-1。
*   它表示下一个字符是否存在。
*   read()方法的返回类型是 int(整数)。

**语法:**

```
public int read()
```

**程序**

```
import java.io.*;
public class Main
{
	public static void main(String[] args)  throws IOException 
	{
	Buffered Reader object2 = new BufferedReader(new InputStreamReader(System.in));
    System. out. println("enter the string");
    char s1 = object2.read();
    System. out. println(s1);
	}
} 
```

### ready()方法

*   它检查输入流是否准备好被读取。
*   Return ready()方法是布尔值。

**语法:**

```
public boolean ready();
```

**程序**

```
import java.io.*;
import java.util.*;
class Main {
    public static void main(String[] args)
    {

        try {

            String str = "Java Language";
            // create reader instance
            Reader reader= new StringReader(str);  

            System.out.println("Is Reader ready "+ "to be read: "+ reader.ready());
            // check whether it is ready or not
            reader.close();
        }
        catch (Exception e) {
            System.out.println(e);
        }
    }
} 
```
