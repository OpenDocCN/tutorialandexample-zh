# PHP 示例

> 原文：<https://www.tutorialandexample.com/php-example/>

我们可以通过在 php 标签中编写源代码来创建一个简单的程序，并用。php 扩展。**语法:**

```
<?php
// your code here
?>
```

Let us take an example of PHP.

```
<!DOCTYPE html>
<html>
<head>
            <title>PHP Program</title>
</head>
<body>
<?php
echo "Hello PHP";           
?>
</body>
</html>
```
