# CSS 选择器:Selenium IDE 中的内部文本

> 原文：<https://www.tutorialandexample.com/css-selector-inner-text/>

**CSS 选择器:内部文本**

内部文本用于使用显示在网页上的 HTML 标签的**字符串模式**来识别和创建 CSS 选择器。

### 内部文本的语法:

```
css=[HTML tag]

 **[:]:-** 用来象征包含方法。

**包含**:包含一个类属性的值。

 **Text** :显示在网页上任何位置的文本，与它的位置无关。

由于简化的语法，内部文本是识别 web 元素最常用的定位技术之一。

```
