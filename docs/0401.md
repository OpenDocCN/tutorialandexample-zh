# Java 中的拓扑排序

拓扑排序主要用于有向无环图(DAG)中顶点的线性排序。Java 中的拓扑排序演示了如何在编程中做顶点的线性排序。这里顶点的线性排序意味着对于有向非循环图的每两个节点 v 和 u，使得 v - > u，节点 v 应该总是在 u 之前
