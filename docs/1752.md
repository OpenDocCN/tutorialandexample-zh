# HTML

<section>标签</section>

> 原文:[https://www.tutorialandexample.com/html-section-tag](https://www.tutorialandexample.com/html-section-tag)

HTML 中的 section 元素用于定义文档中的节。

## Section 标记的语法:

```
<body>
<h1>……….</h1>
<section>
<h2>……….</h2>
<p> …………………………</p>
</section>
<section>
<h2>………..</h2>
<p>…………………………. <p>
</section>
</body>
```

## 默认 CSS 设置

在大多数浏览器中，以下 CSS 值被保存为默认值，用于定义 section 标签:

**例子**

```
section {
  display: block;  
} 
```
