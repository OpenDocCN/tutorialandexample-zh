# PHP While 循环

> 原文:[https://www.tutorialandexample.com/php-while-loop/](https://www.tutorialandexample.com/php-while-loop/)

PHP while 循环用于运行相同的代码块，直到满足某个条件。如果迭代次数未知，应该使用它。

![](../Images/405877c5bdb7f4caf23e2de3580df807.png)

语法:

```
<?php
while (condition){
  //Code to be executed;
}
?>
```

**举例:**

```
<?php
$n= 1;
while($n<=5){
echo "$n<br/>"
$n++;
}
?>
```

**输出**

One

Two

Three

Four

Five
