**X 路径定位器:**

Xpath 用于使用任何**属性**或**可见测试**来识别元素。

**X 路径的语法:-**

```
//html tag [@attribute=’value’]
```

例如- //div [@class='label']

**或**

```
//html tag [text () =’Visible Text’]
```

例如-//div [text () ='Tasks']
