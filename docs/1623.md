# Javascript 打印

> 原文：<https://www.tutorialandexample.com/javascript-print>

JavaScript 帮助使用**窗口**对象的**打印**功能。在 JavaScript 中打印函数 **window.print()** 执行时打印当前网页。

我们可以使用 **onclick** 事件直接调用函数。

### 例子

```
<!DOCTYPE html>  
<html>  
<body>  
<form>  
<input type="button" value="Print" onclick="window.print()" />  
</form>  
</body>  
<html>
```
