# XML 的特性

> 原文：<https://www.tutorialandexample.com/features-of-xml>

*   XML 文件是文本文件，可以由任何文本编辑器管理。
*   它很容易学习和实现，因为它的语法规则不到 10 条。
*   XML 是可扩展的，因为它规定了标签的结构规则。标签上没有任何说明。
*   它非常适合长期数据存储和数据可重用性。
*   它处理只有一个根元素的树结构中的数据。
